## Versioning

`neviri-cli` follows [Semantic Versioning](https://semver.org/):

- **`X.Y.Z`** — major / minor / patch
- **`X.Y.Za`/`b`/`rc`** — alpha / beta / release candidate pre-releases

The Phase milestones map to versions like this:

| Phase | Version | Status |
|---|---|---|
| Phase 1 internal alpha | `0.1.0a1` | shipped 2026-05-10 |
| Phase 2 public beta | `0.9.0b1` | shipped 2026-05-11 |
| Phase 3 GA | `1.0.0` | shipped 2026-05-13 |

From `1.0.0` onward, exit codes, output schemas, and command names are
stable. Breaking changes ship only in a new major version.
